# 🧠 Сайт психолога — Вильданова Аделина

Одностраничный сайт для психолога с дизайном, вдохновленным **Liquid Glass** от Apple.

## 🎨 Концепция дизайна

### Liquid Glass Style
Дизайн вдохновлен новым визуальным языком Apple — эффектом "жидкого стекла":
- Полупрозрачные карточки с размытием фона (backdrop-blur)
- Мягкие градиенты с переливами
- Стеклянные поверхности с легким свечением
- Органичные, плавные формы
- Минималистичная цветовая палитра
- Тонкие рефракционные эффекты света

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   npx serve .
   ```

## 🛠 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — современные возможности (backdrop-filter, градиенты, анимации)
- **Vanilla JavaScript** — интерактивность без фреймворков
- **Google Fonts** — типографика
