## Детальное описание секций

### 1. Navigation (Навигация)

**Тип:** Фиксированная шапка с эффектом стекла

**Содержимое:**
- Логотип / Имя (слева)
- Навигационные ссылки (центр/справа):
  - О психологии
  - С чем работаю
  - Обо мне
  - Стоимость
  - Как я работаю
- CTA кнопка: "Записаться" (справа)

**Поведение:**
- Прозрачная изначально
- Становится "стеклянной" при скролле
- Мобильное меню — бургер

## Интерактивность

### Навигация
- Плавный скролл к секциям при клике на ссылки
- Подсветка активного пункта меню

### Карточки услуг
- Hover-эффект: подъём + усиление тени
- На мобильных: возможно аккордеон для описаний

### CTA кнопки
- Ведут на форму записи (Telegram/WhatsApp/форма)
- Hover-эффект: подъём + изменение тени

### Scroll-анимации
- Появление элементов при скролле
- Каскадное появление карточек
